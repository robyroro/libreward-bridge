# Data flow

| Data | Source | Stored | Destination | Reason |
|---|---|---|---|---|
| Tenant display name/public ID | operator | tenant row | API responses/log context | administration and isolation |
| API key | operator CLI | keyed hash and public prefix only | integrator | service authentication |
| Operator identity/role/key | restricted administrator | account row; keyed hash/prefix; append-only action events | private operator API | cross-tenant operations and accountability |
| Exact amount/currency/description | integrator | reward/provider operation | wallet-core; claim recipient; webhook | execute and audit reward |
| Opaque external reference | integrator | reward | integrator webhook/API | reconciliation without identity |
| Scalar metadata (≤4 KiB) | integrator | reward | tenant API only by default | integration context |
| Claim token | derived from random material + PRF key | token keyed hash and derivation material, never plaintext | recipient link; idempotent create replay | bearer authorization |
| Taler peer-push URI | wallet-core | AES-256-GCM ciphertext | claim recipient | wallet acceptance |
| Wallet transaction ID | wallet-core | provider operation | operator API/log context | reconciliation |
| Webhook secret | generated by Bridge | AES-256-GCM ciphertext | tenant once | delivery authentication |
| Event payload/status | Bridge | events/deliveries | tenant endpoint | lifecycle notification |
| Source IP | network | not in application DB | transient rate-limit memory/log infrastructure | abuse control |
| Wallet balance/pending totals | wallet-core | short-lived sanitized per-currency snapshots | private operator API and Prometheus | fail-closed funding control and alerting |

Logs contain request ID, public tenant/reward IDs where explicitly logged, event type, operation, sanitized code, duration, and retry count. They must not contain bearer paths, authorization, decrypted provider data, or response payloads.
