# Project summary

LibreReward Bridge is a privacy-preserving open-source bridge for distributing exact-value rewards through GNU Taler. It lets an integrator issue an accountless bearer claim while an operator-controlled wallet funds the peer-push operation and the integrator receives a signed lifecycle webhook.

The existing AGPL-3.0-or-later repository is a feasibility prototype, not the proposed funded output: it already contains the TypeScript/Fastify/PostgreSQL core, mock flow, a preliminary wallet-core adapter, concurrency controls, OpenAPI, tests, and sanitized valueless sandbox evidence. It is not approved for production or real money.

The proposed €42,000 project funds future upstream-aligned interoperability, a reproducible valueless testbed, independent security/privacy validation, reusable SDK/reference work, accessibility/operations, and alpha-to-beta maturation. It does not reimburse completed prototype work or fund commercial marketing.
