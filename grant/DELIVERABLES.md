# Deliverables

1. Upstream-aligned wallet provider with capability/version negotiation, restart reconciliation, and documented ambiguous-effect behavior.
2. Valueless GNU Taler interoperability harness with compatibility/failure matrix and sanitized JSON plus Markdown evidence.
3. External security/privacy review package, genuine reviewer output, remediations, and residual-risk register.
4. Tested TypeScript SDK, webhook verifier, OpenAPI conformance suite, generic reference integrator, and upgrade/migration documentation.
5. Accessible claim flow evidence, hardened deployment profiles, backup/restore and incident exercises, contributor onboarding, and alpha/beta release artifacts.
6. Upstream questions, issue/patch links where accepted, and final commons/maintenance report.

All code and documentation will be published under AGPL-3.0-or-later unless a third-party artifact retains its compatible upstream license. No wallet secrets, bearer URIs, personal data, or real-money evidence will be published.
